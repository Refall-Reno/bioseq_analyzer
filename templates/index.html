BioInformatics Project

Analisis GC Content dari File Sekuens DNA.

Aplikasi bioinformatika berbasis web yang dirancang untuk mengevaluasi stabilitas termal dari sampel lingkungan. Pengguna cukup mengunggah data spesimen, kemudian sistem akan secara otomatis menjalankan rangkaian analisis komputasi berikut:

  • Ekstraksi dan pembacaan file FASTA/FASTQ.
  • Kalkulasi frekuensi nukleotida (A, T, G, C).
  • Pemeringkatan kandidat berdasarkan persentase GC tertinggi.
  • Pembuatan visualisasi data dan laporan CSV otomatis.

Input Data Sekuens

Format didukung: .fasta, .fas, .txt, .fastq

Pilih File Spesimen Anda

Catatan Penting: Pastikan file Anda berisi lebih dari satu sekuens (Multi-FASTA) agar sistem dapat membandingkan dan mencari Top 3 kandidat terbaik.
